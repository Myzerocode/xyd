# CodeSample
<Subtitle>
    A component to display code samples with syntax highlighting.
</Subtitle>

## Overview
The `CodeSample` component is used to display code samples with syntax highlighting. It is a wrapper around the [`codehike`](https://codehike.org/) package.

## Example
<CodeSample
    name="helloWorld.js"
    description="Hello World in JavaScript"
    codeblocks={[
        {
            value: "console.log('Hello World');",
            lang: "js",
            meta: "helloWorld.js",
        }
    ]}
    size="full"
/>
