# Introduction


[XYD](#) is a [Static Site Generator](#) (SSG) designed for building fast, content-centric websites.
In a nutshell, XYD takes your source content written in [Markdown](#), applies a theme to it, and generates static HTML pages that can be easily deployed anywhere.

## Use Cases
<Steps>
    <Steps.Item>
        **Documentation Sites**: XYD is great for documentation sites. It's easy to organize content, and the generated pages are fast and easy to navigate.
    </Steps.Item>
    <Steps.Item>
        **Personal Blogs**: XYD is perfect for personal blogs. It's simple to set up, easy to use, and generates fast, static pages.
    </Steps.Item>
</Steps>

## Developer Experience
<Steps>
    <Steps.Item>
        **Simple Configuration**: XYD is designed to be simple to use. You can get started with just a few lines of configuration.
    </Steps.Item>
    <Steps.Item>
        **Fast Development**: XYD has a built-in development server that automatically reloads your changes as you work.
    </Steps.Item>
</Steps>
